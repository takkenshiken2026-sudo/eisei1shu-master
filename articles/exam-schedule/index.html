第一種衛生管理者試験の試験日程【令和8年度・月次センター実施】

第一種衛生管理者試験の日程は、安全衛生技術試験協会（https://www.exam.or.jp/）の最新要項が正本です。令和8年度は全国の試験センターで月次実施され、本番は13:30〜16:30の180分・100問5科目です。たとえば10月13日（日）の回を受けるなら、8月中旬頃から申込受付が始まり、先着順・定員制のため早めの日程確定が重要です。本記事では月次スケジュールの読み方と、申込締切チェックリスト記事への接続を表にまとめます。

この記事の要点

本記事を読むと、令和8年度の月次実施日程と申込2か月前ルールが分かり、希望回を逃さず受験地を確保するための逆算表が作れます。

  • 月次センター実施
  • 13:30〜16:30・100問
  • 2か月前から先着
  • 東京・大阪休日試験
  • 要項が正本

1令和8年度の月次実施スケジュール

令和8年度の第一種衛生管理者試験は、全国の試験センターで月次に実施されます。従来の「年2回（4月・10月）」だけに限定されない年度もあるため、必ず安全衛生技術試験協会の年度日程表で確認してください。

| 項目 | 内容（令和8年度） |
| --- | --- |
| 実施形態 | 全国試験センターで月次 |
| 試験時間 | 13:30〜16:30（180分） |
| 問題数 | 100問・5科目 |
| 休日試験 | 東京・大阪で拡大（要項確認） |

  • たとえば2026年10月13日（日）13:30開始の回を選ぶ場合
  • 8月中旬頃から申込が始まり
  • 定員に達し次第そのセンターは締切になり

複数回受験を予定する場合は、春期・秋期だけでなく月次回も候補に入れてください。

2試験センター別の日程確認

受験地は試験センター単位で日程・定員が異なります。公式サイトの「試験日程・受験地一覧」で、自分の希望都市に該当する回があるかを確認します。

| 確認項目 | 方法 |
| --- | --- |
| センター名 | 要項の受験地一覧 |
| 試験日 | センターごとの実施日 |
| 定員 | 先着順で満席になる |
| 会場住所 | 受験票到着後に照合 |

  • 例えば大阪府内在住で10月回を受ける場合
  • 大阪センターの実施日と定員状況を受付開始日（試験日の約2か月前）に確認し
  • 満席なら近隣センター（兵庫・京都等）を第2希望に設定し

併願記事と組み合わせ、他試験との日程衝突も申込前にチェックしてください。

313:30〜16:30・100問5科目の本番形式

第一種衛生管理者試験の本番形式は、年度を問わず次のとおりが基本です。

| 項目 | 内容 |
| --- | --- |
| 開始 | 13:30 |
| 終了 | 16:30 |
| 時間 | 180分 |
| 問題数 | 100問 |
| 科目 | 5科目（各20問） |

  • 5科目は関係法令（有害業務・有害以外）
  • 労働衛生（有害業務・有害以外）
  • 労働生理

たとえば10月13日（日）試験なら、直前1か月記事の180分リハーサルと同じ時間帯で演習し、合格基準（総合60％・各科目40％足切り）を意識した学習計画に切り替えてください。

4申込2か月前からの逆算表

申込は希望試験日のおおむね2か月前から先着順で始まります。

| 試験日（例） | 申込開始目安 | やること |
| --- | --- | --- |
| 4月13日（日） | 2月中旬 | 春期回の定員確保 |
| 7月13日（日） | 5月中旬 | 夏期回の候補 |
| 10月13日（日） | 8月中旬 | 秋期回の定員確保 |

  • たとえば10月13日（日）13:30〜16:30の回なら
  • 8月15日頃までに受験資格・写真データ・受験料8
  • 800円の支払準備を済ませ
  • 受付開始日当日にログインして第1希望センターを確保し

申込締切前チェックリスト記事の表と照合し、不備なく申込みを完了してください。

5休日試験（東京・大阪）の確認

令和8年度は、東京・大阪の試験場で休日試験が拡大される見込みです。詳細な実施日・会場は年度要項で公表されるため、古い情報の「4月と10月のみ」という表現に頼らず最新案内を確認してください。

| 地域 | 確認ポイント |
| --- | --- |
| 東京 | 休日試験の追加回 |
| 大阪 | 休日試験の追加回 |
| その他 | 月次センター実施 |

例えば首都圏在住で平日が難しい場合、東京の休日試験回が追加されていれば選択肢が広がります。申込み・受験の流れ記事と合わせ、受験票到着後に会場名・開始13:30・100問形式を最終確認してください。

6よくある質問

試験は年に何回実施されますか？
令和8年度は全国試験センターで月次実施が基本です。センターごとに実施日が異なり、定員に達し次第締切になります。たとえば10月13日（日）13:30〜16:30の回のほか、4月・7月など複数回から選べる年度もあります。正確な回数と日程は安全衛生技術試験協会の令和8年度要項で確認してください。
申込はいつから始まりますか？
希望試験日のおおむね2か月前から受付が始まります。例えば10月13日（日）の回なら8月中旬頃から申込可能です。先着順・定員制のため、受付開始日から1週間以内の申込がおすすめです。申込締切前チェックリスト記事で資格・写真・8,800円の決済を事前に準備しておくとスムーズです。
東京・大阪の休日試験とは何ですか？
令和8年度は東京・大阪の試験場で休日試験が拡大される見込みです。平日が取りにくい受験生向けに、日曜等の追加回が設けられる場合があります。たとえば10月13日（日）のような日曜回は、13:30〜16:30・100問5科目で実施されます。実施日・会場の詳細は年度要項の試験日程表が正本です。
