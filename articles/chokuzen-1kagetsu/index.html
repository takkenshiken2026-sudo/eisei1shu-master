第一種衛生管理者・試験直前1ヶ月の対策【やること優先順位】

第一種衛生管理者試験の直前1ヶ月は、新しい教材を増やすより得点できる論点の定着に切り替える時期です。たとえば試験日10月13日（日）13:30〜16:30まで残り4週間なら、100問5科目・全体60％かつ各科目40％以上（足切り）を意識し、演習の誤答解き直しと苦手科目の底上げに週の時間の7割を配分します。本記事では優先順位を週単位の表にまとめます。

この記事の要点

本記事を読むと、残り1か月で何を優先し何を止めるべきかが分かり、科目別足切りで不合格になるリスクを下げる復習計画が立てられます。

  • 残り4週間の優先順位
  • 5科目足切り40％
  • 100問180分
  • 誤答解き直し優先
  • 新教材は控える

1残り4週間の得点戦略

直前1か月の基本方針は「新規インプット最小・既習論点の得点化」です。第一種は100問・180分（13:30〜16:30）で、5科目すべて40％以上が必要です。

| 週 | 重点 |
| --- | --- |
| 第1週 | 5科目の正答率を可視化 |
| 第2週 | 40％未満科目を底上げ |
| 第3週 | 誤答ノート本番形式2回 |
| 第4週 | 直前1週間記事へ接続 |

  • たとえば9月13日時点で10月13日試験まで残り4週間なら
  • 第1週末までに5科目×20問のミニ演習で正答率を記録し
  • 最も低い科目（例：労働生理38％）に週5時間を追加し

25科目足切り40％の見える化

合格基準は総合60％以上かつ各科目40％以上です。100問中、科目ごとの配点は要項・問題構成表で確認してください。

| 科目 | 足切りライン（目安） |
| --- | --- |
| 関係法令（有害業務） | 40％以上 |
| 関係法令（有害以外） | 40％以上 |
| 労働衛生（有害業務） | 40％以上 |
| 労働衛生（有害以外） | 40％以上 |
| 労働生理 | 40％以上 |

例えば関係法令（有害以外）だけ35％の演習結果なら、総合70％でも不合格（足切り）になります。得意科目で点数を稼ぐ前に、最下位科目を40％ラインまで引き上げるのが先です。

3100問180分の時間配分

本番は13:30開始・16:30終了の180分です。100問なら1問あたり平均1.8分、最初の90分で60問を目安に進めると見直し時間が確保しやすくなります。

| 時間帯 | 目標 |
| --- | --- |
| 13:30〜15:00 | 60問まで解答 |
| 15:00〜16:00 | 残り40問＋マーク問題 |
| 16:00〜16:30 | 未記入欄・消し忘れ確認 |

  • たとえば直前1か月の日曜に
  • 13:30から180分タイマーをセットし
  • 100問演習を2回実施して時間感覚を体に覚えさせ

分からない問題はマークして後回しにし、最終10分で未記入をチェックする習慣をつけてください。

4苦手科目の優先順位づけ

5科目のうち演習正答率が最も低い科目から時間を配分します。労働生理は用語・数値の暗記比重が高く、関係法令は条文の主体（事業者・労働者・衛生管理者）の整理が効きます。

| 科目 | 直前1か月のやり方 |
| --- | --- |
| 関係法令 | 頻出条文の主体整理 |
| 労働衛生 | 数値・濃度基準の暗記 |
| 労働生理 | 用語解説＋一問一答 |

  • 例えば労働生理が38％で他科目が55〜60％なら
  • 週5時間を労働生理に配分し
  • 残り2時間で関係法令の誤答解き直しを行い

過去問の解き方記事と組み合わせ、同じ分野の問題を3日後にもう一度解いて定着を確認してください。

5直前1か月でやめること

直前1か月に効果が薄い（または逆効果の）学習は次のとおりです。

1. 新しい問題集・通信講座の第1章から読み始める
2. 得意科目だけ繰り返し、苦手科目を放置する
3. 時間を計らずに問題を解く
4. 徹夜で範囲を詰め込む
5. 模試結果だけ見て誤答分析をしない

  • たとえば9月第2週に未読の章が5章残っている場合
  • 新規インプットは止め
  • 既に解いた演習の誤答20問を本番形式で解き直す方が得点に直結し

残り1週間になったら「直前1週間」記事の最終確認リストに切り替えてください。

6よくある質問

直前1ヶ月で新しい教材は買うべきですか？
原則おすすめしません。残り4週間で新教材を完走するのは現実的でなく、既習範囲の誤答解き直しの方が得点効率が高いです。たとえば10月13日試験まで1か月なら、新規購入より誤答ノート50問の本番形式2回解き直しを優先してください。どうしても弱点補強が必要なら、1科目・1章に絞ってください。
科目別足切りはどう対策しますか？
5科目それぞれの演習正答率を記録し、40％未満の科目を最優先で底上げします。例えば関係法令（有害業務）が38％なら、週3時間をその分野の頻出数値・主体整理に充て、20問演習で45％まで引き上げるのが目標です。合格基準記事の足切り表と照合しながら進めてください。
1ヶ月前から模試は受けるべきですか？
本番形式（180分100問）の演習セットを1〜2回受ける価値はあります。たとえば9月第3週日曜13:30から180分タイマーで100問を解き、5科目の得点を記録して弱点1科目を特定するのが定番です。模試後は誤答理由を「知識不足・時間切れ・ケアレス」に分類し、最後の1週間はケアレス防止の確認リストに集中してください。
