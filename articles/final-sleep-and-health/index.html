第一種衛生管理者試験の直前の睡眠と健康管理【13:30〜16:30向け】

第一種衛生管理者試験の直前——は残り14日から——睡眠を「必須範囲」——に置く方が得点——を守りやすい。100問·180分（13:30〜16:30）·5科目各20問——は後半90分——の集中力——が勝負。たとえば10/1から就寝23:00·起床6:00——（睡眠7時間）——カフェイン15:00以降控える——が定番。正本は安全衛生技術試験協会（https://www.exam.or.jp/）。chokuzen-1shukan——睡眠項目、final-day-checklist——前日就寝——と役割分担。受験料8,800円·2か月前申込。

本記事を読むと、直前2週間の就寝·学習·カフェインの線引き——が分かり、たとえば「D-3に23:30まで演習」——場合の翌日調整——まで把握できます。

  • 睡眠7時間
  • カフェイン15時截止

114日前から就寝23時固定

直前2週間——学習量より就寝時刻——の固定が先。

| 優先 | 控える |
| --- | --- |
| 睡眠7時間前後 | 徹夜·24:00以降演習 |
| 解き直し·誤答ノート | 未読章一括インプット |
| 就寝·起床±30分以内 | カフェイン15:00以降 |
| 弱点2科目だけ | 新規100問大量追加 |

たとえば10/1〜10/12——23:00就寝·6:00起床——14日間——→10/13 13:30開始——時の覚醒。final-week-prep——直前週解き直し——と併用。40％未満科目——は睡眠削らない。

2カフェインと学習90分の上限

カフェインと学習時間——はセット管理。

| ルール | 内容 |
| --- | --- |
| カフェイン | 15:00以降控える |
| 学習上限 | 平日90分·D-7以降45分 |
| 朝食 | 毎日·当日メニューも決める |
| 入浴 | 就寝3時間前21:00目安 |

例えば平日17:00〜18:30——90分演習——→19:00以降——解き直し10問——→22:30就寝。10/10に23:30演習——→翌日45分削減——→22:30就寝復帰。100問·180分——後半の読み取り精度優先。

3前日入浴21時·当日朝の流れ

D-1と当日朝——の健康タイムライン。

| 時刻目安 | 行動 |
| --- | --- |
| D-1 18:00 | 学習終了 |
| D-1 21:00 | 入浴·体温調整 |
| D-1 22:00 | 就寝 |
| 当日 6:00 | 起床·水分補給 |
| 当日 11:00 | 出発 |

たとえば10/12 21:00入浴·22:00就寝·10/13 6:00起床——→11:00出発——→exam-venue-and-region——会場ルート確認。final-day-checklist——前日手順連動。12:00最寄り駅到着——13:00会場入り。

438度発熱時の判断表

体調不良時——の初動を表に。

| 症状 | 判断 |
| --- | --- |
| 38度以上·強い頭痛 | 医師·要項を確認 |
| 37度台·軽い風邪 | 水分·マスク·監督者相談 |
| 軽い不眠 | 深呼吸（徹夜追い込み非推奨） |

例えば試験当日朝37.8度——軽食·水分——→11:00出発——→会場到着後監督者相談。38.5度以上——医師·要項判断。goukaku-kijun——得点以前に体調優先。exam.or.jp——要項正本。

5徹夜禁止の理由（再現性）

直前の徹夜——は暗記量より本番再現性——を下げる。

| 比較 | 結果 |
| --- | --- |
| D-2徹夜50問 | D-1正答率-20％ |
| 睡眠7時間×7日 | 13:30開始時安定 |
| 誤答20問解き直し | 徹夜より効率的 |

たとえばD-2に24:00まで50問——→D-1の13:30模擬——正答率低下——→1回記録——で徹夜誘惑減。chokuzen-1shukan——睡眠項目参照。100問·5科目——後半40問——の集中力優先。8,800円·2か月前申込。

6よくある質問

直前に徹夜して暗記を増やしてもよいですか？
非推奨。睡眠を削るより——既存の間違い20問——解き直し——の方が13:30開始時——の読み取り精度——を保ちやすい。たとえばD-2に23:00就寝を守れなかったら——D-1は学習15分のみ·22:00就寝——に戻す。final-day-checklist——前日手順。100問·180分·各科目8/20毎回。
体調が悪いときは受験を続けますか？
38度以上の発熱·めまい·強い腹痛——は無理に続けず係員へ相談。具体例として、当日朝37.5度——水分·軽食·マスク——→12:00到着後——監督者へ体調申出。受験可否——要項·会場案内正本。exam-venue-and-region——会場連絡先確認。延期——saijuken-guide参照。
独学でも睡眠管理はできますか？
はい。就寝23:00·起床6:00——を14日間カレンダー登録——だけで始められる。たとえば学習タイマー——22:30切断——カフェイン15:00以降紅茶——の2ルール。前日——final-day-checklist——当日——time-limit-strategy——配分参照。100問·60％+40％——goukaku-kijun確認。
