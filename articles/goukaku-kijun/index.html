第一種衛生管理者の合格点・合格基準【100問中何問？科目別40％足切り】

第一種衛生管理者試験の合格基準は、総合60％以上かつ各科目40％以上（足切り）です。問題数は100問（5科目×各20問）で、180分（13:30〜16:30）の筆記試験です。「44問中何問」という表現は誤りで、正しくは100問中60問以上が総合合格ラインの目安です。たとえば10月13日（日）試験まで残り4週間の演習で100問中70問正解（70％）でも、労働生理だけ35％（7/20問）なら足切りで不合格になります。本記事では60％・40％基準を数値例と表で整理します。

この記事の要点

本記事を読むと、100問中何問取れば合格か、5科目の足切り40％がどう効くかが具体例で分かり、演習得点の目標設定ができます。

  • 100問・5科目
  • 総合60％以上
  • 各科目40％足切り
  • 44問は誤り
  • 演習で可視化

1100問中何問で合格か（60％基準）

第一種衛生管理者試験は100問出題され、総合得点率60％以上が合格条件の一つです。

| 項目 | 数値 |
| --- | --- |
| 問題数 | 100問 |
| 合格ライン（総合） | 60％以上＝60問以上 |
| 試験時間 | 180分（13:30〜16:30） |
| 配点 | おおむね1問1点 |

たとえば100問中62問正解（62％）なら総合基準はクリアします。しかし各科目40％足切りも同時に必要なため、62問でも1科目が39％なら不合格です。「44問中何問」という情報は別試験や誤記の可能性があるため、100問基準で学習計画を立ててください。

25科目各20問と40％足切り

5科目はそれぞれ20問出題され、各科目40％以上（8問以上）が必要です。

| 科目 | 出題数 | 足切り（40％） |
| --- | --- | --- |
| 関係法令（有害業務） | 20問 | 8問以上 |
| 関係法令（有害以外） | 20問 | 8問以上 |
| 労働衛生（有害業務） | 20問 | 8問以上 |
| 労働衛生（有害以外） | 20問 | 8問以上 |
| 労働生理 | 20問 | 8問以上 |

例えば関係法令（有害業務）で20問中10問正解（50％）なら足切りはクリア、7問正解（35％）なら不合格ラインです。得意科目で総合70％を取っても、1科目でも39％以下なら足切り不合格になる点に注意してください。

3足切り不合格の具体例

足切りで不合格になるパターンを数値例で示します。

| ケース | 総合 | 最低科目 | 結果 |
| --- | --- | --- | --- |
| A | 70％（70/100） | 労働生理35％（7/20） | 不合格 |
| B | 58％（58/100） | 全科目42％以上 | 不合格（総合不足） |
| C | 65％（65/100） | 全科目40％以上 | 合格 |

ケースAは、関係法令・労働衛生で高得点でも労働生理7/20問（35％）が足切りに引っかかります。

  • たとえば10月13日試験まで残り4週間なら
  • 演習で労働生理20問の正答率を記録し
  • 8問以上（40％）を先に確保してください

4得点の見積もり方

演習得点から合格可能性を見積もる手順です。

1. 5科目×20問の演習で科目別正答数を記録
2. 各科目の正答率が40％（8問）以上か確認
3. 100問合計の正答率が60％（60問）以上か確認
4. 40％未満科目に週5時間を追加配分
5. 2週間後に同じ演習セットを解き直し伸びを確認

  • 例えば最新の100問模擬で関係法令（有害業務）12/20（60％）
  • 関係法令（有害以外）9/20（45％）
  • 労働衛生（有害業務）10/20（50％）
  • 労働衛生（有害以外）8/20（40％）
  • 労働生理7/20（35％）
  • 合計46/100（46％）なら
  • 総合・労働生理の両方が不足しているため
  • 労働生理を最優先で底上げし

5演習での合格ライン確認

本番前に演習で合格ラインを確認する習慣をつけます。

| 演習タイプ | 確認項目 |
| --- | --- |
| 科目別20問 | 8問以上（40％） |
| 100問フル | 60問以上（60％） |
| 180分制限 | 時間配分（1.8分/問） |

たとえば9月第3週日曜13:30から180分タイマーで100問を解き、5科目の得点を表に記録します。40％未満の科目があれば直前1か月記事の優先順位に従い、週5時間をその科目に追加してください。100問・60％・40％足切りの3点セットを、試験日10月13日（日）まで毎回の演習で確認するのが定番です。

6よくある質問

100問中何問取れば合格ですか？
総合得点率60％以上、すなわち100問中60問以上が目安です。ただし各科目40％以上（20問中8問以上）の足切りも同時に必要です。たとえば65/100問（65％）でも、労働生理7/20問（35％）なら足切りで不合格になります。総合と科目別の両方を演習で毎回確認してください。
44問中何問という情報は正しいですか？
第一種衛生管理者試験は100問出題であり、44問という配点は誤りです。おそらく別資格試験や古い情報との混同です。正しくは100問中60問以上（60％）が総合合格ラインの目安で、5科目それぞれ20問中8問以上（40％）が足切りラインです。安全衛生技術試験協会の要項・問題構成表が正本です。
足切りはどの科目でも適用されますか？
はい。5科目すべてに40％以上が必要です。関係法令（有害業務・有害以外）、労働衛生（有害業務・有害以外）、労働生理のいずれか1科目でも39％以下なら不合格です。たとえば関係法令（有害業務）20問中7問（35％）だけでも、他科目がすべて50％以上でも足切り不合格になります。演習で最下位科目を特定し、8問以上を先に確保してください。
