第一種衛生管理者試験の難易度の誤解【足切り40％·公表50％前後】

第一種衛生管理者試験の難易度誤解——は公表合格率50％前後——の使い回しと足切り40％——の軽視が典型。100問·180分（13:30〜16:30）·5科目各20問·総合60％+各科目40％——の3条件——を同時に満たす必要がある。たとえば10月13日（日）試験まで残り8週の9/7（日）100問——65/100·労働生理7/20——なら総合合格ライン超え——でも足切り不合格相当。44問表記——は100問——難易度判断の前提訂正。exam-difficulty·goukakuritsu——記事と役割分担。

この記事の要点

本記事を読むと、難易度の5誤解——と総合·科目別——の2軸評価——が分かり、たとえば「9/7 65/100·生理7/20→9月生理週2×20問」——まで設計できます。

  • 公表は参考

1誤解1：公表50％前後＝簡単

合格率だけ——では判断不可。

| 指標 | 用途 |
| --- | --- |
| 公表50％前後 | 全体傾向 |
| 自分の100問得点 | 合格判断 |
| 各科目8/20 | 足切り |

たとえば公表52％の年度——初回100問45％·生理35％——なら統計より厳しい。goukakuritsu——公表率の読み方。exam-difficulty——3軸評価。44問表記——は100問——前提訂正。

2誤解2：「3科目だけ」で足切り回避

5科目すべて——40％以上が必要。

| 誤解 | 正 |
| --- | --- |
| 3科目集中 | 5科目バランス |
| 得意だけ | 最下位8/20先 |
| 44問 | 100問 |

例えば法令2科目——得意でも——生理7/20——で不合格。9/7（日）100問——5科目分けて記録。benkyou-junban——5科目順。10月13日試験まで残り6週なら弱点科目週2×20問。

3誤解3：総合65％＝合格確定

総合と足切り——は別判定。

| 得点例 | 結果 |
| --- | --- |
| 65/100·全科目8/20+ | 合格圏 |
| 65/100·生理7/20 | 足切りNG |
| 58/100·全科目9/20+ | 総合要底上げ |

たとえば9/14（日）100問——68/100·最低7/20——→翌週その科目40問。goukaku-kijun——2軸表正本。口コミ「44問30」——→100問中60+8/20——に読み替え。

4誤解4：SNS短期合格の再現性

短期合格論——は条件未記載。

  • SNS：3ヶ月
  • 現実：80〜120時間
  • 演習：100問記録
  • 足切り：8/20
  • 本番：13:30·180分
  • 44問：誤情報

例えばSNS「1ヶ月合格」——→6/14 100問48％——なら3〜4ヶ月計画——へ切替。study-hours-myth——時間神話。difficulty-for-beginners——初学者基準。演習得点——で難易度測定。

5exam-difficulty·goukakuritsu接続

難易度3記事の分担です。

| 記事 | 役割 |
| --- | --- |
| 本記事 | 難易度の誤解 |
| exam-difficulty | 3軸·足切り脱落 |
| goukakuritsu | 公表率·独学ライン |
| goukaku-kijun | 60％+40％計算 |

たとえば7月——goukakuritsu——公表資料確認——→8月exam-difficulty——パターン分析——→9/7 100問——自分の数字優先。100問·180分·5科目——exam.or.jp正本。10月13日試験まで残り8週なら9/1から100問月2回。

6よくある質問

「簡単」という情報は信じてよいですか？
SNS「簡単」「3ヶ月」——は未検証。9/7（日）100問——5科目記録——が正本。たとえば52/100·1科目7/20——なら8〜12週計画——が現実的。exam-difficulty——3軸評価。44問表記——は100問——要項確認。
難易度は実務経験者と初学者で同じですか？
演習得点の伸び方——が異なる。経験者——は16週×8時間——初学者——は80〜120時間——目安。たとえば同じ9/7 100問——でも58％と42％——では配分を変える。difficulty-for-beginners——初学者版。各科目8/20——は共通前提。
44問と難易度の関係は？
44問——は旧情報·混同。難易度の正本——は100問——60％+各40％足切り——goukaku-kijun——計算例。たとえば「44問中30」——→100問中60+8/20——に読み替え。616問——14回バンク·本番100問。exam.or.jp要項——毎年確認。
