第一種衛生管理者・労働生理の勉強法【計算・WBGT・8/20足切り】

第一種衛生管理者試験の労働生理は100問中20問・180分中約36分——計算と用語が半々程度。足切り8/20（40％）未満は再受験者の最多原因科目の一つ。たとえば10月13日試験まで残り10週の9月なら9/7労働生理20問7/20（35％）——なら9月後半は100問より労働生理20問×2＋公式メモ——を優先。温熱・WBGT・作業強度を公式1枚に集約してください。

この記事の要点

本記事を読むと、労働生理20問の学習順・計算練習・8/20ラインの上げ方が分かります。たとえば7/20→9/20→10/20と2週で40％ライン到達する型が使えます。

  • 20問・8問
  • WBGT計算
  • 公式メモ
  • 足切り最多
  • 36分目安

1労働生理20問の出題構造

20問の内訳目安です。

| タイプ | 問数目安 |
| --- | --- |
| 計算（WBGT等） | 6〜8問 |
| 用語・生理 | 6〜8問 |
| 事例 | 4〜6問 |

たとえば9/7 20問中計算3/8・用語4/8——計算優先復習。100問180分で労働生理36分（20問×1.8分）が目安。10月13日試験まで残り8週なら8/31までに20問×2回40％以上を確認。

2公式メモ1枚の作り方

計算問題用のメモテンプレです。

| 公式 | 用途 |
| --- | --- |
| WBGT | 温熱 |
| 作業強度 | エネルギー代謝 |
| 酸素消費量 | 作業評価 |

  • 例えば8/10（日）公式メモ作成
  • 8/17計算10問時間制限なし
  • 8/24（日）計算10問2.5分/問——公式暗記記事と併用

WBGT5問を週2回×4週で計20セット反復が目安。

| ペア | 比較軸 |
| --- | --- |

たとえば9/7混同5問——比較表1枚→9/14（日）解き直し3/5。netchu-wbgt-guide記事と一問一答WBGT10問/日×5日。残り2問は9/21まで再反復。

47/20から8/20への2週集中

40％未満からの標準リカバリーです。

| 日 | 内容 |
| --- | --- |
| 1〜3日目 | テキスト温熱章 |
| 4〜7日目 | 計算10問×2 |
| 8〜10日目 | 20問解き直し |
| 14日目 | 再20問10/20 |

例えば9/7 7/20→9/8〜9/14の1週集中→9/21（日）9/20（45％）——100問延期とセット。2週で8/20未満なら10月13日試験を1回後ろに。

5180分演習での時間配分

本番労働生理の時間目安です。

| 問題 | 時間 |
| --- | --- |
| 計算1問 | 2.5分 |
| 用語1問 | 1.5分 |
| 20問合計 | 約36分 |

たとえば9/14（日）180分100問で労働生理未着手8問——10月13日試験まで残り4週なら未着手3問以下を目標。時間計測演習記事の科目別配分と一致。計算6問は2.5分/問で先に着手する習慣を。

6よくある質問

労働生理は最後でいい？
学習順は最後でも、8/20足切りチェックは早期に。たとえば8/7 20問7/20（35％）——を8月第2週に検出。100問通し前に20問×2回40％以上——を確認。9月100問で7/20のまま——は不合格リスク大。
計算が苦手な場合は？
公式メモ10回声出し→同类型5問×4セット——が最短。たとえば9/15（火）WBGT5問を2.5分以内×10回——から20問演習へ移行。計算ドリル記事の短時間反復と併用。8/20到達まで100問は延期。
field-rodoseiri記事との関係は？
本記事は20問・足切り・学習全体の進め方。field-physio-*は頻出論点・計算詳細——の深掘り。たとえば9/7 20問後、弱点がWBGTならfield-physio-calculation＋netchu-wbgt——へ接続。
