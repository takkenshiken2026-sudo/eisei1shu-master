第一種衛生管理者試験の最初の30日の学習計画

第一種衛生管理者試験の最初の30日は、要項確認・教材決定・2科目の入口学習・演習40問で現在地を測る期間です。たとえば2026年6月5日（木）に開始するなら7月4日（金）までに、関係法令（有害以外）と労働衛生（有害以外）を中心に週10時間×4週＝40時間を確保します。30日終了時点で5科目それぞれ最低8問ずつ触れ、足切り40％に届いていない科目を第2ヶ月の重点に据えます。

この記事の要点

本記事を読むと、独学開始から30日間で何をどの順番で終えるべきかが週単位で分かり、2ヶ月目以降の計画修正に使える初期データが得られます。

  • 30日・週10時間
  • 要項→テキスト→演習
  • 2科目入口
  • 5科目ミニ演習
  • 第2ヶ月へ接続

1第1週（Day1〜7）：要項と環境づくり

第1週は学習量より準備を優先します。

| Day | タスク | 時間目安 |
| --- | --- | --- |
| 1〜2 | 要項・受験資格・試験日 | 2時間 |
| 3 | テキスト候補3冊を比較 | 1時間 |
| 4 | 学習場所・時間帯を決定 | 30分 |
| 5〜7 | テキスト第1章開始 | 6時間 |

たとえば6月5日開始なら6月11日までに10月13日試験の申込開始日（8月中旬）をカレンダー登録し、テキスト1冊を決定します。この週は演習10問以上が目標で、正答率より「毎日触れた日数」を記録してください。

2第2週（Day8〜14）：テキスト第1章完走

第2週は関係法令（有害以外）のテキスト第1章を完走します。

| 日 | 内容 | 演習 |
| --- | --- | --- |
| 月〜木 | 各45分テキスト | 各2問 |
| 土 | 3時間まとめて | 10問 |
| 日 | 2時間解き直し | 誤答5問 |

例えば6月12日〜18日の週で演習15問、正答率50％なら第3週も同分野を継続。60％超なら労働衛生（有害以外）の第1章へ移行します。章を跨ぐ前に誤答ノート10問を解き直してください。

3第3週（Day15〜21）：第2科目の入口

第3週は労働衛生（有害以外）を追加し、関係法令は週2時間の解き直しのみにします。

| 配分 | 時間/週 | 内容 |
| --- | --- | --- |
| 新規 | 6時間 | 労働衛生（有害以外） |
| 復習 | 2時間 | 関係法令の誤答 |
| 演習 | 2時間 | 計20問/週 |

たとえば6月19日〜25日に労働衛生10問・関係法令解き直し10問、労働衛生が42％・関係法令が55％なら計画どおりです。30％未満の科目が出たら第4週の新規を止め、解き直しに週5時間振り替えてください。

4第4週（Day22〜30）：5科目ミニ演習

第4週は5科目それぞれ8問（計40問）のミニ演習でバランスを確認します。

| 科目 | 8問の目標 | 40％ライン |
| --- | --- | --- |
| 関係法令（有害業務） | 3問以上 | 40％ |
| 関係法令（有害以外） | 4問以上 | 50％ |
| 労働衛生（有害業務） | 3問以上 | 40％ |
| 労働衛生（有害以外） | 4問以上 | 50％ |
| 労働生理 | 3問以上 | 40％ |

  • 例えば7月1日〜4日に40問を解き
  • 労働生理2/8問（25％）だけ低ければ
  • 8月までに労働生理週3時間を初学者計画に追加し

30日で全科目60％は不要ですが、どの科目が足切りリスクかを把握することが目的です。

530日終了時の振り返りシート

30日終了時に次のシートを埋め、第2ヶ月計画へ接続します。

| 項目 | 記入例 |
| --- | --- |
| 開始日 | 6/5 |
| 総学習時間 | 38時間 |
| 最低正答率科目 | 労働生理28％ |
| 第2ヶ月重点 | 労働生理＋有害業務 |
| 申込予定日 | 8/15 |

たとえば7月4日時点で38時間・40問演習済みなら、初学者6か月計画の月2配分（有害業務2科目）へ進めます。30時間未満しか確保できていない場合は、試験日を1回後ろにずらすか週次時間を12時間へ増やす判断をしてください。

6よくある質問

30日で全体の何割を終えるべきですか？
5科目すべてに触れ、2科目を第1章レベルで理解できれば十分です。100問中60問レベル（60％）は30日時点では不要です。たとえば40問演習で総合45％・労働生理30％なら、「第2ヶ月は労働生理を厚く」という判断材料になります。完璧主義で範囲を広げすぎるより、誤答解き直しの質を優先してください。
30日目までに申込すべきですか？
必須ではありませんが、希望試験日の2か月前を過ぎると定員リスクが上がります。例えば10月13日試験なら8月中旬が申込開始目安です。30日目（7月初旬）時点では資格・写真・8,800円の準備状況を確認し、申込締切チェックリスト記事で8月の申込完了を目標にすると安全です。学習ペースが週6時間未満なら試験日を後ろにずらす検討も有効です。
テキストは30日で何章まで？
テキスト全体の2〜3割（第1章〜第3章相当）が目安です。5科目すべてを1冊でカバーするテキストなら、30日で目次の20％に印を付けられれば計画どおりです。たとえば第1章のみ完走で15％なら、第2章半ばまで第5週で進めるペースに修正してください。演習40問と解き直しを優先し、未読章の先読みは控えめに。
