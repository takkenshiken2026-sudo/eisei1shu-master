第一種衛生管理者試験の仕事と勉強の両立【境界線と休息】

第一種衛生管理者試験の受験者は社会人が多く、仕事と勉強の「境界」が曖昧だと両方の質が下がります。100問5科目・120時間前後の独学は、平日21:00〜22:30・日曜13:30演習——のように時間帯を固定し、残業日は10問最小セットに切り替える。たとえば10月13日試験まで残り14週、週10時間確保・睡眠7時間維持——が両立の定番。本記事では境界線・休息・家族調整を表にまとめます。

この記事の要点

本記事を読むと、仕事に勉強時間が侵食されないルール設計と、繁忙期の縮小版が分かります。たとえば残業22時帰宅週は翌朝15分10問のみ——と切り替えられます。

  • 21:00境界
  • 日曜13:30
  • 残業週10問
  • 睡眠7h
  • 家族共有

1時間帯の境界線

仕事と勉強の切り分け例です。

| 時間帯 | 用途 |
| --- | --- |
| 〜21:00 | 仕事・家事 |
| 21:00〜22:30 | 演習10問 |
| 日曜13:30〜16:30 | 100問180分 |
| 就寝前15分 | 用語5語 |

たとえば火木21:00デスク着席、日曜13:30タイマー開始——を16週カレンダー固定。21:00前に演習を始めると仕事の集中と勉強の質が両方低下します。

2残業週のフォールバック

残業・出張週の縮小ルールです。

| 状況 | 勉強 |
| --- | --- |
| 通常週 | 10h |
| 残業2日/週 | 6h（10問×4日） |
| 出張週 | 3h（一問一答） |
| 連続2週6h未満 | 試験日見直し |

例えば9/7週22時帰宅3日——朝15分10問×3＋日曜4h解き直し。ゼロ週にしない。2週連続3hなら10/13→2月回へ——社会人向け計画記事で再計算。

3睡眠と演習量のトレードオフ

睡眠6時間未満が続く週は演習を減らします。

| 睡眠 | 演習 |
| --- | --- |
| 7h以上 | 通常 |
| 6〜7h | 80％ |
| 6h未満×3日 | 50％ |
| 徹夜後 | 休み or 10問のみ |

たとえば9/14週残業で睡眠6h×4日——週10h→5h、100問延期。徹夜で100問演習は正答率低下とケアレス増——直前睡眠記事どおり休息優先。

4家族・職場との調整

両立のための共有例です。

| 相手 | 伝える内容 |
| --- | --- |
| 家族 | 日曜13:30は演習 |
| 上司 | 火木21時以降は非残業希望 |
| 同僚 | 10/13受験・協力依頼 |

例えば9月初旬に「日曜午後と火木夜は勉強」と家族共有——10/6最終模試も同枠。急な残業は最小10問で「空けない1週」を維持してください。9/14週に家族へ次週日曜13:30演習を再確認すると当日のトラブルが減ります。

5両立成功の週次チェック

毎週日曜5分で両立度を確認します。

| 項目 | OK基準 |
| --- | --- |
| 実績時間 | 6h以上 |
| 睡眠平均 | 6.5h以上 |
| 仕事重大ミス | 0 |
| 演習 | 10問以上 |

たとえば9/14週：7h・睡眠6.8h・演習25問——両立OK。6h・睡眠6.0h·演習0——なら翌週フォールバック。12週中3週未達なら試験日延期を検討してください。

6よくある質問

仕事のストレスで勉強できない週は？
仕事優先1週間、勉強は最小10問。ストレス期に100問追加は両方のパフォーマンスを下げます。たとえば9/7週プロジェクト繁忙——9/14週に20問解き直しで回復。2週連続繁忙なら10/13受験を見直し、モチベーション記事のゴール延期と併用してください。
在宅勤務は両立しやすいですか？
通勤時間分（往復60分）を一問一答20問に充てやすい反面、仕事と勉強の境界が曖昧になりやすいです。21:00アラームでデスク切替——が在宅向け。たとえば12:30昼休み15分10問、21:00本番10問——と時間帯を分けてください。
両立と合格率の関係は？
週6時間×20週＝120時間——を睡眠7h維持で確保できれば、週15時間突撃より合格率が高い傾向。100問5科目40％足切りは理解の定着が必要で、睡眠削って時間を増やすより週8時間×15週の方が9/28に62/100到達しやすいです。
