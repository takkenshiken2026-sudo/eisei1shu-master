第一種衛生管理者試験の形式【100問5科目・四択・180分の時間配分】

第一種衛生管理者試験の形式は、100問・180分（13:30〜16:30）・5科目（各20問）の筆記・マークシート試験です。出題はおおむね四択問題で、合格は総合60％以上かつ各科目40％以上です。たとえば10月13日（日）本番まで残り4週なら、9月第3週日曜13:30から180分タイマーで100問を解き、科目ごとに36分（180÷5）の時間配分を試すのが定番です。正本は安全衛生技術試験協会（https://www.exam.or.jp/）の要項で、本記事では形式・時間配分・マークシートの実務に特化します。

この記事の要点

本記事を読むと、100問5科目の構成と四択マークシートのルール、180分の科目別時間配分が分かり、本番形式の演習設計にそのまま使えます。

  • 100問・5科目
  • 四択マークシート
  • 180分・13:30開始
  • 科目36分目安
  • 要項が正本

1100問5科目の問題配分

第一種衛生管理者試験は5科目から合計100問が出題されます。

科目出題数内容の例
関係法令（有害業務）20問化学物質・濃度基準
関係法令（有害以外）20問労基法・安全衛生法全般
労働衛生（有害業務）20問粉じん・有機溶剤等
労働衛生（有害以外）20問騒音・照明・作業環境
労働生理20問体温・心拍・作業負荷

たとえば100問フル演習では、科目順が本番と異なる場合でも各20問ずつ解き、40％足切り（8問以上）を科目ごとに確認します。44問出題という古い情報は誤りで、100問が正しい配分です。

2四択マークシートの記入ルール

解答はマークシートにHB鉛筆で記入します（詳細は要項・当日案内で確認）。

項目注意点
鉛筆HB（当日持ち物記事参照）
選択方式四択1問1答が基本
消し跡きれいに消す（消しゴム必須）
未記入提出前に全問チェック

例えば9月第2週の20問演習でも、HB鉛筆で解答欄をマークシート風に塗りつぶす練習をすると本番に慣れます。10月13日（日）試験当日は開始直後に100問すべてに目を通し、見落としやすい後半20問の時間を意識的に確保してください。

3180分の科目別時間配分

180分を5科目で均等に割ると、おおむね36分/科目が目安です。

時間帯科目（例）目安
13:30〜14:06関係法令（有害業務）36分・20問
14:06〜14:42関係法令（有害以外）36分・20問
14:42〜15:18労働衛生（有害業務）36分・20問
15:18〜15:54労働衛生（有害以外）36分・20問
15:54〜16:30労働生理36分・20問

たとえば9月第3週日曜13:30から180分タイマーで100問を解き、各20問終了時点の時刻を記録すると、苦手科目で時間オーバーしていないかが分かります。最後の10分は見直し専用に確保するのが定番です。

4出題形式と解答の注意点

出題形式はおおむね次のパターンです（年度で詳細は要項確認）。

形式特徴対策
四択1問1答最も多い消去法で絞る
数値問題濃度・温度等頻出数値リスト
条文・制度関係法令条文番号の当たり
組合せ正しいものを選ぶ1つずつ検証

例えば関係法令（有害業務）の20問演習で、許容濃度0.5ppmと5ppmの四択が並ぶ場合、単位と桁数まで確認しないと誤答になります。100問演習でケアレスミスが3問以上ある場合は、時間制限なしで10問だけ丁寧に解き直すリハーサルも有効です。

5本番形式演習の進め方

試験形式に慣れるには、次のサイクルが効果的です。

1. 13:30開始・180分タイマーで100問を解く
2. 5科目の得点と正答数を表に記録
3. 40％未満科目を翌週優先で復習
4. 2週間後にもう一度100問本番リハーサル
5. 直前1か月記事の優先順位に接続

たとえば10月13日（日）試験の2週前（9月29日）に100問演習で総合68％・全科目9問以上を達成できれば、形式面の準備はおおむね完了です。当日の持ち物記事でHB鉛筆・消しゴム・受験票を前日までに準備し、会場では13:00入りを目安にしてください。

6よくある質問

試験時間と問題数は？
180分（13:30〜16:30）で100問です。5科目それぞれ20問ずつ出題されます。1問あたり平均1.8分ですが、科目によって難易度が異なるため36分/科目を目安に進めます。たとえば10月13日（日）本番では、13:30開始・16:30終了を前提に月1回以上の本番リハーサルがおすすめです。
マークシートは何色の鉛筆ですか？
HB鉛筆が基本です（年度要項・当日の案内で最終確認）。消しゴムも必須で、消し跡が残ると読み取りエラーの原因になります。例えば演習でもHB鉛筆で解答欄を塗る練習をすると、本番のマークシート記入に慣れます。当日の持ち物記事で前日までに筆記用具を揃えてください。
科目の順番は固定ですか？
問題冊子の科目順は年度・問題用紙で確認してください。いずれにせよ5科目×20問＝100問です。演習では科目順にこだわらず、各20問ずつ40％足切り（8問以上）を確認する習慣が重要です。たとえば180分演習で後半2科目に時間が足りない場合、36分/科目の配分を見直し、苦手科目を前半に回す戦略も検討してください。
