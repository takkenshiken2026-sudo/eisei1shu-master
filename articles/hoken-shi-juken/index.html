保健師·看護師から第一種衛生管理者【受験·勉強のコツ】

保健師·看護師など医療系資格保有者は、健康診断·感染症·人間工学——で理解が早い一方、労働安全衛生法の選任要件·化学物質·粉じんの法令体系——は初見になりやすい。第一種衛生管理者試験は100問·180分（13:30〜16:30）·5科目各20問·総合60％+各科目40％足切り·受験料8,800円——正本は安全衛生技術試験協会（https://www.exam.or.jp/）。たとえば10月13日（日）試験まで残り16週の6/14（土）独学開始なら7月健診論点復習→8月法令20問——が定番。

この記事の要点

本記事を読むと、医療系出身者の強みを活かしつつ法令·有害業務の弱点をどう補強するかが分かります。たとえば9/7（日）100問62/100でも関係法令（有害）7/20——なら翌週半分をkankei-hourei-benkyou——演習に回せます。

  • 医療系強み
  • 法令弱点
  • 5科目順
  • 100問
  • 40％足切り

1医療系受験生の強みと弱点

資格背景ごとの傾向です。

| 分野 | 傾向 |
| --- | --- |
| 健診·感染症 | 理解速い |
| 人間工学 | 復習短時間可 |
| 選任·数値 | 初見多い |
| 化学·粉じん | 法令体系要学習 |

たとえば6/14（土）独学開始——保健師なら健診章——は1週短縮。10月13日試験まで残り16週なら7月に法令以外20問初回。field-eisei-other-basicsと併用。各科目8/20足切りを毎回記録。

医療系向けの時間配分です。

| 科目 | 配分目安 |
| --- | --- |
| 法令（有害以外） | 週2時間 |
| 衛生（有害以外） | 週1.5時間 |
| 法令（有害） | 週2.5時間 |
| 衛生（有害） | 週2.5時間 |
| 労働生理 | 週2時間 |

例えば7/7（日）benkyou-junban順——法令以外20問9/20——→7/14衛生20問10/20——と週1科目。健診強みは衛生以外——で時短。13:30〜16:30·180分を意識。

6よくある質問

看護師は法令が苦手、どこから？
安衛法骨格章読了後——関係法令（有害以外）20問——から。たとえば6/21（土）テキスト第1章後——20問9/20——が正常範囲。40％未満なら2週解き直し。benkyou-junban記事と同順。10月13日試験まで残り14週なら7月末まで法令40問初回完了。100問·180分·5科目足切り。
健診は得意、時間を削っていい？
短縮可だが——ゼロは危険。健診4〜6問/20——は頻出。たとえば9/7（日）健診5/5——でもストレス2問誤答——なら20問9/20未満。kenko-shindan-guide——で3日10問/日反復。field-eisei-other-basics——とセット。各科目8/20足切りを確認。
benkyou-junbanとの違いは？
benkyou-junban——は全受験生向け5科目順——本記事は医療系の強み·弱点配分。たとえば6/14開始——junban記事で順序把握——→本記事で健診時短·法令+2時間/週。100問通しは9月以降。受験料8,800円·2か月前申込。exam.or.jp要項で資格確認。
