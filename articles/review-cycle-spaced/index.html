第一種衛生管理者試験の間隔を空けた復習【7日→3日→1日サイクル】

第一種衛生管理者試験の復習は、同じ問題を詰め込むより間隔を空ける方が定着率が上がります。定番は「7日後→3日後→1日前」の3段階解き直しです。たとえば9月7日に誤答8問を記録したら、9/14（7日後）→9/17（3日後）→10/12（1日前）に同じ8問を解き直す——5科目40％足切りに届いていない科目の誤答を優先します。

この記事の要点

本記事を読むと、間隔復習のカレンダーの作り方と、5科目のどの誤答を優先的に解き直すかが分かります。たとえば9/7誤答8問→9/14に6/8正解→9/17に残り2問、10/12最終確認という7日→3日→1日サイクルが組めます。

  • 7日→3日→1日
  • 間隔復習
  • 誤答優先
  • 5科目連動
  • 直前は解き直しのみ

17日→3日→1日サイクルの基本

間隔復習の3段階です。

段階タイミング目的
17日後忘却確認
23日後再定着
3試験1日前最終確認

たとえば9/7誤答8問→9/14解き直し6/8正解→9/17に残り2問→10/12最終確認、という流れ。7日後の正答率が50％未満なら、テキスト該当章を挟んでから3日後リトライしてください。過去問解き直しサイクル記事と役割分担が近いため、どちらか一方の表に統合しても構いません。

25科目別の復習優先度

復習時間が限られるときの優先順位です。

優先度条件復習対象
最高40％未満科目の誤答全件7日後
高再誤答2回以上3日後リトライ
中混同理由の誤答比較表＋解き直し
低ケアレス1回のみ本番形式で対応

例えば労働生理35％・誤答リスト15問なら、週8時間中4時間を労働生理の7日後解き直しに。関係法令60％で誤答2問のみなら、7日後解き直しは10分で済ませ、時間を労働生理へ回します。

3週次復習カレンダーの例

9月第2週の復習カレンダー例です。

曜日復習内容
月7日後リスト（前週誤答）
水3日後リスト（再誤答）
土新規演習20問＋当日誤答記録
日7日後リスト（土曜誤答は翌週日）

たとえば9/14（月）に9/7誤答8問、9/16（水）に9/13再誤答2問、9/17（土）に新規20問——と固定すると抜けが減ります。社会人は土日に7日後リストをまとめて処理する型も有効です。

4間隔復習と100問通しの両立

100問通し演習週は復習量を調整します。

週のタイプ復習新規
通常週7日後20問演習20問
100問週7日後10問のみ100問通し
直前2週再誤答のみ100問×2

例えば9/21週は180分100問演習のため、7日後解き直しは10問に絞る。100問翌週9/28に、9/21誤答分の7日後解き直しを優先処理。新規と復習の合計が週12時間を超える場合は、新規を減らして復習を優先してください。

5直前1週間の間隔復習

試験1週間前は新規誤答の追加を止めます。

1. 再誤答2回以上の問題20問だけ解く
2. 5科目の頻出数値リストを15分/日確認
3. 試験2日前に7日→3日→1日の最終ラウンド
4. 前日は新規演習ゼロ・軽い見直しのみ

たとえば10月13日試験なら10/6〜10/12は再誤答20問＋数値確認、10/12夜は誤答ノートの再誤答行のみ15問。直前1週間記事の持ち物・会場確認と並行し、徹夜で範囲追加は避けてください。

6よくある質問

7日間隔は必須ですか？
厳密に7日でなくても構いません。5〜9日の範囲で「一度忘れた頃」に解き直すのが要点です。たとえば土曜演習なら翌週土曜解き直し、という曜日固定の方が続けやすい人もいます。間隔より「必ず解き直す日をカレンダーに入れる」ことが重要です。
復習と新規演習、どちらを優先？
40％未満科目がある間は復習優先です。例えば労働生理35％なら、新規20問より7日後解き直し15問を先に処理。5科目すべて45％以上で安定したら、新規100問通しの比重を上げてください。直前1か月は復習9：新規1が目安です。
間隔復習と間違いノートの違いは？
間違いノートは「記録の置き場」、間隔復習は「いつ解き直すかのスケジュール」です。ノート1行に7日後・3日後・前日の日付を書き、間隔復習記事のサイクルどおり実行する——とセットで使うのが効率的です。過去問解き直しサイクル記事は100問単位の復習に寄り、本記事は誤答1問単位の間隔管理に寄ります。
