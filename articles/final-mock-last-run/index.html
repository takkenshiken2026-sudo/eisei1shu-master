第一種衛生管理者試験の直前模試の受け方【180分100問·10/5最終】

第一種衛生管理者試験の直前模試は、残り14日から逆算して1回だけ本番形式で受けるのが目安です。100問·180分（13:30〜16:30）·5科目×各20問·総合60％+各科目40％足切り。たとえば10/5（日）13:30〜16:30で100問通し→法令有害12/20·生理9/20を記録→10/6〜10/12は弱点2科目へ90分/日、が定番。受験料8,800円。模試の活用法記事は実施·記録·分析の通年手順、本記事は直前最終1回の日程·切り替えに特化します。例えば6/11（木）·10/13（日）試験まで残り18週でも10/5（日）最終模試後は新規100問停止→解き直しのみ→直前1週間の学習計画記事へ、が定番です。

この記事の要点

本記事を読むと、直前模試をいつ·どう受け、5科目別正答数をたとえば「生理9/20」のように記録して最終7日に反映する手順が表で分かり、模試の活用法記事との使い分けも確認できます。

  • 10/5最終1回
  • 5科目記録
  • 弱点2科目60％
  • 翌日解き直し

記録5科目別·時間切れ·ケアレス

たとえば10/5（日）13:30開始·16:30終了で100問5科目通し→62/100·生理9/20を記録。10/6以降は新問題集は開きません。以降は解き直しと直前1週間の学習計画記事へ接続してください。

5模試の活用法記事との2列比較表

通年の模試実施と直前最終1回は2記事で分担します。模試の活用法記事は月2回·記録表·24時間誤答分析、本記事は10/5頃の最終180分100問1回·翌日からの切り替えを説明します。

例えば8月〜9月模試の活用法記事で月2回180分実施後、10/5（日）本記事で最終1回→10/6（月）直前1週間の学習計画記事、の順が定番です。2記事を混同すると通年記録だけで最終1回の日程が未定になり、または最終模試だけで月次分析·記録表が抜けます。模試日程記事で5回カレンダーも併用。

6よくある質問

直前模試は何回受ければよいですか？
本番形式180分100問は試験2週前に1回が目安。たとえば10/13（日）試験なら10/5（日）最終→10/6〜10/12は解き直し·確認のみ。2回目以降は同セット解き直し、新規100問は非推奨。模試の活用法記事の月2回記録と併用。各科目8/20毎回記録。
模試で60％未満だったら延期しますか？
総合と科目別を分けて判断します。たとえば10/5（日）55/100だが全科目9/20+なら総合不足→10/6〜10/10解き直し40問→再判定。2科目40％未満なら11月回延期を検討。合格基準記事で2軸を確認。直前範囲絞り込み記事で範囲も絞り。
本記事と模試の活用法記事の違いは？
模試の活用法記事はいつ·どう記録·分析するかの通年手順、本記事は10/5頃の最終180分100問1回の日程·切り替えです。たとえば9月模試の活用法記事で月2回実施→10/5（日）本記事→10/11（土）試験直前1週間の対策記事、の順。100問·180分·13:30〜16:30共通。試験協会要項で毎年確認。
