第一種衛生管理者試験の直前模試の受け方【180分100問·10/5最終】

第一種衛生管理者試験の直前模試——は残り14日から逆算——して1回だけ——本番形式で受ける——のが目安。100問·180分（13:30〜16:30）·5科目各20問·総合60％+各科目40％足切り。たとえば10/5（日）13:30〜16:30——100問通し——→法令有害12/20·生理9/20——記録——→10/6〜10/12——弱点2科目へ90分/日——が定番。mock-exam-how-to——は実施·記録·分析、本記事——は直前最終1回の日程。正本は安全衛生技術試験協会（https://www.exam.or.jp/）。受験料8,800円·2か月前申込。

6よくある質問

直前模試は何回受ければよいですか？
本番形式180分100問——は試験2週前——に1回——が目安。たとえば10/13試験——10/5最終——→10/6〜10/12——解き直し·確認——のみ。2回目以降——同セット解き直し——新規100問——は非推奨。mock-exam-how-to——月2回記録——と併用。各科目8/20毎回。
模試で60％未満だったら延期しますか？
総合——と科目別——を分けて判断。たとえば10/5——55/100だが——全科目9/20+——→総合不足——→10/6〜10/10——解き直し40問——→再判定。2科目40％未満——→11月回延期——検討。goukaku-kijun——2軸正本。final-scope-narrowing——範囲絞り。
final-mock-last-runとmock-exam-how-toの違いは？
mock-exam-how-to——いつ·どう記録·分析——するか——の通年手順。本記事——10/5頃——最終180分100問——1回——の日程·切り替え。たとえば9月——mock-exam-how-to——月2回——→10/5本記事——→10/11 chokuzen——の順。100問·180分·13:30〜16:30共通。exam.or.jp——要項確認。
