第一種衛生管理者・過去問の効果的な回し方【1周目・2周目・3周目】

第一種衛生管理者試験の過去問は、解いた回数より「各周回の目的」で効率が変わります。本番は100問・180分（13:30〜16:30）・5科目各20問で、総合60％かつ各科目40％以上の足切りがあります。一衛マスターに掲載の過去問616問（14回分相当）は演習バンクとして使い、たとえば10月13日試験まで残り12週なら1周目（科目別20問）→2周目（解き直し）→3周目（100問通し）の順が定番です。本記事では周回ごとの目的と記録表をまとめます。

この記事の要点

本記事を読むと、過去問を何周・どの順番で回せば5科目足切りに備えられるかが分かり、たとえば1周目45％→2周目55％→3周目65％の伸びを記録する型が使えます。

  • 1〜3周目で目的分担
  • 100問・180分
  • 5科目足切り
  • 誤答3分類
  • 616問は演習バンク

11周目：理解と弱点洗い出し

1周目の目的は「正答率」より弱点科目の特定です。時間制限なしで解き、解説を読んで理解を深めます。

| 項目 | 1周目のルール |
| --- | --- |
| 問題数 | 科目別20問×5科目 |
| 時間 | 制限なし（解説必読） |
| 記録 | 科目別正答数 |
| 目標 | 40％未満科目を特定 |

たとえば9月第1週に関係法令（有害）20問を解き、12/20（60％）なら次科目へ。労働生理14/20（70％）でも、関係法令（有害以外）7/20（35％）なら2周目は当該科目優先です。一衛マスターの過去問は分野タグで科目別に絞り込めます。

22周目：誤答の解き直しと定着

2周目は1周目の誤答を1週間後に解き直します。同じ問題を「覚えているつもり」で外さないことが合格の分岐点です。

| ステップ | 内容 |
| --- | --- |
| 1 | 1周目誤答リストを作成 |
| 2 | 7日後に同じ問題のみ解く |
| 3 | 再誤答は用語解説→テキスト該当章 |
| 4 | 正答率50％以上で次セットへ |

  • 例えば1周目で40問中22問正解（55％）なら
  • 2周目は18問の誤答を解き直し
  • 14問以上正解（誤答の78％）を目標にし

間違いノート記事のフォーマットと併用すると、再誤答の理由が蓄積しやすくなります。

33周目：100問・180分の本番形式

3周目以降は本番条件に近づけます。13:30開始・180分・100問5科目で通し演習します。

| 時間帯 | 目標 |
| --- | --- |
| 13:30〜15:00 | 60問まで |
| 15:00〜16:00 | 残り40問 |
| 16:00〜16:30 | 見直し・未記入確認 |

たとえば9月第3週日曜13:30から180分タイマーで100問を解き、総合68/100（68％）・最低科目労働生理9/20（45％）を記録。8問未満科目があれば4周目は当該科目20問の解き直しを優先し、100問通しは月1回に固定してください。

4616問演習バンクの使い方

一衛マスター掲載の616問（14回分相当）は、「全部解く」より科目・テーマで絞る使い方が効率的です。

| フェーズ | 616問の使い方 |
| --- | --- |
| 初期 | 科目別20問×5 |
| 中期 | 弱点科目40問集中 |
| 直前 | 100問通し＋解き直し |

例えば616問を一度に回す必要はなく、9月時点で200問解いて5科目すべて40％以上なら十分なことが多いです。出題傾向記事（616問分析）と併せ、頻出テーマから優先して演習量を確保してください。

5周回ごとの記録表テンプレート

次の表をスプレッドシートまたはノートにコピーして使います。

| 周回 | 日付 | 形式 | 総合 | 最低科目 | 次の一手 |
| --- | --- | --- | --- | --- | --- |
| 1周目 | 9/7 | 科目別20問 | 55％ | 35％ | 2周目準備 |
| 2周目 | 9/14 | 誤答18問 | 78％ | 50％ | 3周目へ |
| 3周目 | 9/21 | 100問180分 | 68％ | 45％ | 労働生理追加 |

たとえば3周目で68％でも労働生理9/20（45％）なら、4周目は100問通しより労働生理20問の解き直しを優先。10月13日試験まで残り3週なら、100問通し2回＋弱点科目解き直しが現実的な上限です。

6よくある質問

過去問は何周すれば足りますか？
最低3周（理解→解き直し→本番形式）が目安です。100問通し演習は試験前に2〜3回できれば十分なことが多いです。たとえば616問すべてを3周する必要はなく、弱点科目を含め200〜300問を質高く回す方が効率的です。各周回で40％未満科目が残る場合は周回数を増やす前に、当該科目のテキスト復習を挟んでください。
1周目から時間を計るべきですか？
1周目・2周目は時間制限なしがおすすめです。理解と誤答分析が目的のため、180分制限は3周目以降に移します。例えば9月第1週は解説を読みながら40問に3時間かけても問題ありません。時間配分の練習は「時間を計った演習」記事の180分100問で行い、1周目との目的を混同しないでください。
44問と100問、どちらで演習しますか？
現在の第一種試験は100問出題です。44問という数字は旧情報や別試験との混同が多いため、演習は100問（または科目別20問）で設計してください。サイト掲載616問は複数回分の演習バンクであり、1回の本番が44問という意味ではありません。合格基準記事の100問・60％・40％足切りと併せて確認してください。
