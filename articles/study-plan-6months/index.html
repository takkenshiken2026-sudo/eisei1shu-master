第一種衛生管理者試験の6ヶ月学習計画【4月開始・10月受験】

第一種衛生管理者試験の6ヶ月学習計画は、4月開始・10月13日（日）13:30試験を想定したゆとりある独学向けスケジュールです。100問5科目・180分・合格60％/40％足切り、総時間130〜150時間・週5〜6時間が目安。たとえば4/7要項確認から始め、7月末テキスト完了→9月100問→10月模試——の月次区切りで進めます。仕事しながらでも継続しやすいペースです。

この記事の要点

本記事を読むと、6ヶ月で何月に何をするかが月次表で分かり、各月末の出口条件も確認できます。たとえば7/31時点で5科目テキスト完了、8/31科目別初回40％以上——というチェックポイントが使えます。

  • 4月開始
  • 週5〜6時間
  • 130〜150時間
  • 月次出口
  • 10月受験

| --- | --- | --- |
| 4月 | 要項・教材・第1章 | 4h |

  • 10月試験は維持可能か要項で申込期限を確認してください

2各月の出口条件

月末に次の条件をチェックします。

| 月末 | 出口条件 |
| --- | --- |
| 4/30 | 要項・テキスト・演習環境 |
| 7/31 | テキスト5科目1周 |
| 8/31 | 5科目初回20問40％以上 |
| 9/30 | 100問180分60％・全8問以上 |
| 10/6 | 模試65％以上 |

例えば8/31時点で労働生理35％なら、9月前半は100問より労働生理解き直し。9/30未達なら10/13受験はギリギリ——模試58/100以下なら延期も検討してください。

3申込タイミング（6ヶ月計画）

希望試験日の2か月前から先着順申込です。

| 試験日 | 申込開始目安 |
| --- | --- |
| 10/13 | 8月中旬〜 |
| 2月回 | 12月〜 |
| 4月回 | 2月〜 |

たとえば10/13受験なら8/10頃に申込準備、受験料8,800円（非課税）を確保。8/31時点で科目別40％未満が2科目あれば、申込前に10月回継続か2月回へ変更を判断。申込チェックリスト記事と併用してください。

46ヶ月と3ヶ月の使い分け

残り期間で計画を選びます。

| 残り | 推奨計画 |
| --- | --- |
| 6ヶ月以上 | 本記事（6ヶ月） |
| 3〜5ヶ月 | 3ヶ月計画 |
| 2ヶ月未満 | 初学者向け＋30日 |

例えば6月開始なら6ヶ月計画の7〜10月部分を圧縮（週8時間）して実行。4月開始なら本記事どおり週5時間で十分。1年計画から6ヶ月に短縮する場合は8月以降を3ヶ月計画の内容に差し替えてください。

56ヶ月計画の週次ルーティン例

週5時間の配分例（5〜9月）です。

| 曜日 | 内容 | 時間 |
| --- | --- | --- |
| 火 | テキスト1章 | 1h |
| 木 | 演習10問 | 1h |
| 土 | 演習20問＋解説 | 2h |
| 日 | 用語・解き直し | 1h |

たとえば8月は土曜20問を科目別初回、9月は土曜180分100問に差し替え。週5時間未満が4週続くなら10月試験を2月回へ1回延期する余裕が生まれます。

6よくある質問

6月から6ヶ月計画は使えますか？
10月試験には4ヶ月しかなく、6ヶ月計画の後半（8〜10月）を3ヶ月計画相当の週8時間で圧縮する形になります。たとえば6/7開始なら7月テキスト集中、8月20問→9月100問→10月模試——4〜7月分を省略または短縮。3ヶ月学習計画記事の方がそのまま使いやすいです。
6ヶ月で週3時間しか取れない場合は？
総時間78時間（26週×3h）となり、初学者120時間に届きません。1年学習計画へ切り替えるか、10月試験を2月回以降に設定してください。例えば週3時間×52週＝156時間なら1年計画で10月翌年受験が現実的です。無理に10月受験すると40％足切りリスクが高まります。
テキストが7月末に終わらない場合は？
8月をテキスト延長に充て、9月から20問初回、10月100問——と1ヶ月ずつ後ろ倒し。8/31時点でテキスト未完了なら10/13受験は10月100問180分が1回のみ——と厳しい状態。模試58/100未満なら2月回へ延期を推奨します。
