第一種衛生管理者試験の間違いノートの作り方【5科目対応フォーマット】

第一種衛生管理者試験の間違いノートは、「どの問題を間違えたか」より「なぜ間違えたか」の記録が合格の鍵です。5科目足切りがあるため、科目タグと40％ラインとの対応も書き込みます。たとえば9月第1週の20問演習で8問誤答なら、ノート8行（科目・理由・関連用語・解き直し日）を当日中に作成し、9月第8週に解き直す——という1週間サイクルが基本です。

この記事の要点

本記事を読むと、5科目対応の間違いノートフォーマットと、再誤答を減らす解き直しの仕方が分かります。たとえば9/7の20問で8問誤答→9/14に6/8正解、再誤答2問は9/17リトライ——という記録の型が使えます。

  • 誤答理由3分類
  • 5科目タグ
  • 7日後解き直し
  • 再誤答優先
  • 40％連動

1間違いノート1行フォーマット

1問1行で次の5項目を記録します。

| 列 | 記入例 |
| --- | --- |
| 日付 | 9/7 |
| 科目 | 労働生理 |
| 問題ID | 演習#042 |
| 誤答理由 | 混同（WBGTと実効温度） |
| 解き直し日 | 9/14 |

たとえば9月7日の演習#042を誤答したら、当日中に1行追加。関連用語は用語解説ページURLまたは用語名をメモし、解き直し日9/14をカレンダーに登録します。ノートアプリでも紙でも、「検索できる」形式を選んでください。

2誤答理由の3分類

誤答理由は次の3つに分類します。

| 分類 | 意味 | 対策 |
| --- | --- | --- |
| 知識不足 | 論点を知らない | テキスト該当章 |
| 混同 | 似た概念の取違え | 比較表・用語 |
| ケアレス | 読み違え・消し忘れ | 見直し習慣 |

例えば20問中8問誤答の内訳が知識不足4・混同3・ケアレス1なら、テキスト復習4問分＋比較表3組を優先。ケアレス1問は解き直しより本番形式演習で対策します。誤答原因分析記事と併用すると、分類のブレが減ります。

35科目別の記録の見方

科目ごとに誤答行数を週次でカウントします。

| 科目 | 9/7週誤答数 | 40％ライン | 優先度 |
| --- | --- | --- | --- |
| 関係法令（有害） | 2 | 60％ | 低 |
| 関係法令（有害以外） | 3 | 45％ | 中 |
| 労働衛生（有害） | 2 | 50％ | 中 |
| 労働衛生（有害以外） | 1 | 55％ | 低 |
| 労働生理 | 5 | 35％ | 高 |

たとえば労働生理だけ誤答5行・正答率35％なら、翌週の演習半分を労働生理の解き直しに充てます。間違いノートの行数が多い科目＝足切りリスク科目、として優先順位を決めるのが5科目試験のコツです。

41週間後の解き直し手順

解き直し日に同じ問題だけを解きます。

1. ノートから当日分の誤答リストを抽出
2. 時間制限なしで1問ずつ解く
3. 正解→ノート行にチェック
4. 再誤答→色付け＋用語解説→3日後再挑戦
5. 再誤答率30％超ならテキスト該当章を読む

例えば9/14に9/7の8問を解き直し、6問正解（75％）なら合格ライン。2問再誤答は9/17に3日後リトライ。間隔を空けた復習記事の7日→3日→1日サイクルと接続してください。

5紙とアプリの選び方

紙ノートとアプリ、どちらでも構いません。

| 方式 | 向く人 |
| --- | --- |
| 紙 | 書くことで記憶定着 |
| スプレッドシート | 科目別集計したい |
| 演習アプリのブックマーク | その場で記録 |

たとえばスプレッドシートなら科目列でフィルタし、労働生理の未解き直し行だけ表示。試験2週前は「再誤答2回以上」の行だけ20問に絞って最終解き直しを行います。ブックマーク復習記事も参照してください。

6よくある質問

全部の誤答をノートに書きますか？
はい。1問1行が原則です。「覚えているから省略」は再誤答の原因になります。ただしケアレス1問だけの場合は、ノート行を作りつつ解き直しは本番形式演習に回しても構いません。20問中8問誤答なら8行必須、40問中5問誤答なら5行——誤答数＝ノート行数を目安にしてください。
解き直しでまた間違えたら？
再誤答行に色付けし、用語解説→テキスト該当章→3日後再挑戦、の順で優先度を上げます。同じ問題を3回連続で誤答したら、関連用語5語をフラッシュカード化し、毎日5分声出し確認を1週間続けてください。再誤答が多い科目は40％足切りリスクが高い信号です。
ノートはいつまで使いますか？
試験前日まで使います。直前1週間は新規誤答の追加を最小限にし、「再誤答2回以上」の行だけ20問に絞って解き直します。たとえば10月6日時点で未処理行が30あっても、試験2週前は頻出20問に絞る方が得点効率が高いです。直前1か月記事の優先順位と整合させてください。
