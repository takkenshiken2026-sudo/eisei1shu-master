第一種衛生管理者・試験当日の持ち物【HB鉛筆・受験票・13:30開始】

第一種衛生管理者試験当日は、13:30〜16:30の180分・100問5科目です。持ち物不備や到着遅れは得点以前のリスクになるため、前日までにHB鉛筆・消しゴム・受験票をそろえ、12:00までに最寄り駅到着・13:00までに会場入りが目安です。たとえば10月13日（日）試験なら、前日22:00就寝・当日7:00起床・11:00自宅出発で、多くの会場に余裕を持って到着できます。正本は安全衛生技術試験協会（https://www.exam.or.jp/）の要項と受験票記載です。

この記事の要点

本記事を読むと、当日必要な持ち物と禁止物品、12:00到着・13:30開始のタイムラインが一覧化され、会場での焦りや不備を防げます。

  • HB鉛筆・消しゴム
  • 受験票必須
  • 12:00到着目安
  • 13:30開始
  • 禁止物品確認

1持ち物チェックリスト（HB鉛筆・消しゴム）

試験当日に必要な持ち物の基本リストです（詳細は要項・受験票で確認）。

| 持ち物 | 数量 | 備考 |
| --- | --- | --- |
| 受験票 | 1通 | 申込時の氏名と一致 |
| 身分証 | 1点 | 本人確認用（要項確認） |
| HB鉛筆 | 2本 | 予備を用意 |
| 消しゴム | 1個 | 消し跡が残らないもの |
| 腕時計 | 1個 | スマホの代わりに |

たとえば10月12日（土）の夜に持ち物をテーブルに並べ、受験票・HB鉛筆2本・消しゴムをバッグに入れてから就寝します。当日朝に「受験票忘れ」が最も多いミスのため、玄関に置いて出発前に最終確認してください。

2受験票と本人確認の注意

受験票は試験日の2〜3週前頃に届く目安です（年度で変動）。

| 確認項目 | 照合先 |
| --- | --- |
| 氏名 | 身分証 |
| 試験日 | 申込時の選択日 |
| 開始時刻 | 13:30 |
| 会場名・住所 | 試験協会の会場案内 |

  • 例えば10月13日（日）13:30開始の受験票なら
  • 9月下旬到着時に会場住所を地図で確認し
  • 10月8日（水）までに最寄り駅から徒歩ルートを検証し

氏名の旧字体不一致があると入場に支障が出るため、申込内容と身分証を前日に照合してください。

312:00到着・13:30開始のタイムライン

当日の動きの目安は次のとおりです。

| 時刻 | 行動 |
| --- | --- |
| 7:00 | 起床・朝食 |
| 11:00 | 自宅出発（例） |
| 12:00 | 最寄り駅到着 |
| 13:00 | 会場入り |
| 13:30 | 試験開始（100問） |
| 16:30 | 試験終了 |

たとえば10月13日（日）で会場まで移動60分なら、10:00出発で12:00到着に余裕を持たせます。開始5分前に席についていれば焦らずマークシート記入に集中できます。試験形式記事の180分時間配分も頭に入れておくと安心です。

4会場での過ごし方と禁止物品

会場では試験監督員の指示に従い、禁止物品を持ち込まないよう注意します。

| 禁止・制限の例 | 理由 |
| --- | --- |
| スマートフォン | 不正防止・集中 |
| 参考書・資料 | 持ち込み不可 |
| 電卓（不要な場合） | 要項で確認 |
| 食事（会場による） | 休憩場所の案内に従う |

例えば10月13日（日）13:30開始の会場では、スマートフォンを電源オフのまま指定場所に預けます。試験中は100問マークシートに集中し、不明点は監督員に手を挙げて確認してください。詳細な禁止物品リストは年度要項と当日の案内が正本です。

5試験終了後の提出確認

16:30終了後は、答案提出を確認してから退室します。

1. マークシートの未記入欄がないか最終チェック
2. 氏名・受験番号の記入漏れがないか確認
3. 監督員の指示どおり答案を提出
4. 預けた持ち物（スマホ等）を受け取り
5. 会場外で結果を待たずに帰路へ

たとえば10月13日16:25頃に後半20問を見直し、16:30に提出して退室する流れです。合格発表までの待ち期間は、誤答ノートの整理や直前1週間記事の振り返りに充ててもよいでしょう。再受験する場合は受験料8,800円・申込2か月前ルールを要項で確認してください。

6よくある質問

持ち物で忘れがちなものは？
受験票・HB鉛筆・消しゴムです。前日に持ち物をテーブルに並べ、バッグに入れてから就寝すると忘れが減ります。たとえば10月13日（日）試験なら、10月12日22:00までに受験票・HB鉛筆2本・消しゴム・身分証を確認してください。会場で貸し出しがない場合があるため、予備のHB鉛筆も用意します。
何時までに会場に入ればよいですか？
12:00までに最寄り駅到着、13:00までに会場入りが目安です。試験は13:30開始・16:30終了の180分です。例えば移動60分の会場なら10:00出発で余裕を持てます。初めての会場ほど試験5日前にルート確認を済ませ、当日の遅延リスクを下げてください。
スマートフォンは持ち込めますか？
会場では電源オフのまま預けるのが基本です（年度要項・当日案内で確認）。試験中は使用できません。時間管理には腕時計を使い、13:30〜16:30の180分を100問に配分します。たとえば試験中にスマホを見る必要はなく、試験形式記事の36分/科目の配分を頭に入れておくとよいでしょう。
